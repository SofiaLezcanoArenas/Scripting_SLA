# Documentación
En este repositorio se podrán encontrar todas las actividades del curso de Scripting desarrolladas de manera individual.
## Ejercicios diagnósticos
Organizados por subcarpetas con nombres que indican su contenido se encuentran:
- **Ejercicio1_fibonacciPrimos**: Contiene la implementación de una función que imprime los números primos de la serie de Fibonacci hasta el n-ésimo término.
- **Ejercicio2_Formatohhmmss**: Implementa una función que recibe una cantidad entera de segundos y retorna un string que muestra esa cantidad en formato HH:MM:SS
- **Ejercicio3_chance**: Permite ingresar una cifra de 4 dígitos para un chance e informa si se es ganador o si recibe algún premio o no.
- **Ejercicio4_POO**: Contiene la implementación de una clase abstracta y dos subclases que sobreescriben métodos para imprimir mensajes de maneras diferentes.
